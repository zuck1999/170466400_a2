Web Design

Course description- This course introduces students to basic web design using HTML and CSS. Throughout the course students are introduced to planning and designing effective web pages, implementing them as well as, enhancing web pages with the use of page layout techniques, text formatting, graphics, images, and multimedia, while ultimatetly producing a functional, multi-page website.
